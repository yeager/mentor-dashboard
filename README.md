# Mentor Dashboard

Debian Mentors dashboard for package review and sponsorship tracking.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install mentor-dashboard
```
